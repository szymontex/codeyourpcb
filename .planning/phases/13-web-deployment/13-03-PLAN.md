  <action>
    1. Create viewer/src/url-state.ts:
       ```typescript
       interface ViewState {
         layers: string[];      // e.g. ['top', 'bottom', 'ratsnest']
         zoom: number;
         panX: number;
         panY: number;
       }

       export function encodeViewState(state: ViewState): string {
         const params = new URLSearchParams();
         params.set('l', state.layers.join(','));
         params.set('z', state.zoom.toFixed(2));
         params.set('x', Math.round(state.panX).toString());
         params.set('y', Math.round(state.panY).toString());
         return '?' + params.toString();
       }

       export function decodeViewState(): ViewState | null {
         const params = new URLSearchParams(window.location.search);
         if (!params.has('l')) return null;
         return {
           layers: (params.get('l') || '').split(',').filter(Boolean),
           zoom: parseFloat(params.get('z') || '1'),
           panX: parseFloat(params.get('x') || '0'),
           panY: parseFloat(params.get('y') || '0'),
         };
       }
       ```
       Use short parameter names (l, z, x, y) to keep URLs compact.

    2. In main.ts init():
       - On startup (after WASM loads), call decodeViewState(). If non-null, apply the returned viewport settings (zoom/pan) and layer visibility to the initial state.
       - Add a "Share" button to toolbar (or add keyboard shortcut Ctrl+Shift+S). On click:
         - Build ViewState from current viewport and layer checkboxes
         - Call encodeViewState(), construct full URL
         - Copy to clipboard via navigator.clipboard.writeText()
         - Show status: "Share URL copied!"
       - Guard share button with `!isDesktop()` — sharing via URL is web-only.

    Do NOT encode design file content in URL — only view state.
    Do NOT change URL on every viewport change (no history.replaceState on pan/zoom).
