<task type="auto">
  <name>Task 1: Create desktop integration module</name>
  <files>viewer/src/desktop.ts</files>
  <action>
Create `viewer/src/desktop.ts` that handles desktop-specific behavior:

1. **`isDesktop(): boolean`** - Detect if running in Tauri: check `window.__TAURI_INTERNALS__` exists.

2. **`initDesktop()`** - Initialize desktop integrations (only call if `isDesktop()` is true):
   - Import `{ listen }` from `@tauri-apps/api/event` and `{ invoke }` from `@tauri-apps/api/core`
   - Listen for `"menu-action"` events with payload string:
     - `"file.new"` -> clear current design (dispatch custom event or call existing function)
     - `"file.open"` -> call `handleOpenFile()`
     - `"file.save"` -> call `handleSaveFile()`
     - `"file.save_as"` -> call `handleSaveFileAs()`
     - `"view.zoom_in"`, `"view.zoom_out"`, `"view.fit"` -> dispatch corresponding UI actions
     - `"view.theme"` -> toggle theme (call ThemeManager or dispatch event)
     - `"help.about"` -> show about dialog (simple alert or custom dialog)

3. **`handleOpenFile()`** - Invoke Tauri `open_file` command:
   - `const result = await invoke<{path: string, content: string} | null>('open_file')`
   - If result is not null, load the content into the viewer (dispatch a custom event or directly call the WASM engine's load function)
   - Store the current file path for subsequent save operations

4. **`handleSaveFile()`** - Save current file:
   - If current file path is known, invoke `save_file` with path and current content
   - If no current path, fall through to `handleSaveFileAs()`

5. **`handleSaveFileAs()`** - Save with file picker:
   - Invoke `save_file_as` with current content
   - Update stored file path with the returned path

6. **Module-level state:** Track `currentFilePath: string | null` for save operations.

Use dynamic imports for `@tauri-apps/api` modules so they don't break web builds (wrap in try/catch or conditional import).
  </action>
  <verify>
TypeScript compiles without errors. The module exports `isDesktop` and `initDesktop`.
  </verify>
  <done>Desktop module handles all menu events, file open/save via Tauri IPC, and detects Tauri environment.</done>
</task>
