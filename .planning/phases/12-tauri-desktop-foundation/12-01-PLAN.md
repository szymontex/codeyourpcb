- Package name: `cypcb-desktop`

- Dependencies: `tauri` (v2, features: `["devtools"]`), `tauri-plugin-dialog` (v2), `tauri-plugin-fs` (v2), `tauri-plugin-window-state` (v2), `serde` (workspace), `serde_json` (workspace)

**src-tauri/build.rs:**
- Just `tauri_build::build()`

**src-tauri/tauri.conf.json:**
- Use `$schema: "https://schema.tauri.app/config/2"`
- productName: "CodeYourPCB", version: "0.1.0", identifier: "com.codeyourpcb.app"
- build.beforeDevCommand: "npm run dev" with cwd "../viewer"
- build.beforeBuildCommand: "npm run build" with cwd "../viewer"
- build.devUrl: "http://localhost:4321"
- build.frontendDist: "../viewer/dist"
- app.windows: single window, title "CodeYourPCB", maximized: true, minWidth: 800, minHeight: 600
- bundle.active: true, targets: "all"
- bundle.icon: ["icons/32x32.png", "icons/128x128.png", "icons/128x128@2x.png", "icons/icon.icns", "icons/icon.ico"]
- bundle.fileAssociations: ext ["cypcb"], mimeType "application/x-codeyourpcb", description "CodeYourPCB Design File"

**src-tauri/capabilities/default.json:**
- identifier: "default"
- description: "Default permissions"
- windows: ["main"]
- permissions: ["core:default", "dialog:default", "fs:default", "window-state:default"]

**src-tauri/src/main.rs:**

  <action>
**viewer/vite.config.ts:**
Update the existing config to be Tauri-compatible:
- Add `host: process.env.TAURI_DEV_HOST || '0.0.0.0'` to server config (replace hardcoded '0.0.0.0')
- Add `watch: { ignored: ['**/src-tauri/**'] }` to server config to prevent infinite rebuild loops
- Add `envPrefix: ['VITE_', 'TAURI_ENV_*']` at top level
- Update build.target to: `process.env.TAURI_ENV_PLATFORM === 'windows' ? 'chrome105' : process.env.TAURI_ENV_PLATFORM ? 'safari13' : 'esnext'` (esnext for web-only, safari13/chrome105 for Tauri)
- Add `minify: !process.env.TAURI_ENV_DEBUG ? 'esbuild' : false` to build config
- Add `sourcemap: !!process.env.TAURI_ENV_DEBUG` to build config

**viewer/package.json:**
- Add `@tauri-apps/api` (v2) to dependencies (not devDependencies)
- Add `@tauri-apps/cli` (v2) to devDependencies
- Add scripts: `"tauri": "tauri"`, `"dev:desktop": "tauri dev"`, `"build:desktop": "tauri build"`
- Keep existing scripts unchanged for web-only development
  </action>
