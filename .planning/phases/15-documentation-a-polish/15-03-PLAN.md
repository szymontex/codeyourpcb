**docs/architecture.md (DOC-09):**
Document the codebase architecture:

1. **System Overview:**
   - CodeYourPCB is a code-first PCB design tool
   - Rust backend compiled to WASM, TypeScript/Vite frontend
   - 14 Rust crates in a Cargo workspace

2. **Crate Dependency Diagram** (ASCII art):
   Show how crates depend on each other. Read each crate's Cargo.toml to build accurate dependency graph.

3. **Crate Descriptions** (for each of the 14 crates):
   - Purpose and responsibility
   - Key types/exports
   - Dependencies on other project crates
   - Approximate size/complexity

   Crates to document:
   - cypcb-parser: Tree-sitter grammar, AST nodes
   - cypcb-core: Core types, units, coordinate system
   - cypcb-world: ECS board model (components, nets, zones)
   - cypcb-drc: Design rule checking engine
   - cypcb-export: Gerber/drill file export
   - cypcb-render: WebGL rendering, WASM entry point
   - cypcb-lsp: Language server protocol implementation
   - cypcb-library: Library management, SQLite storage, FTS5 search
   - cypcb-kicad: KiCad .kicad_mod S-expression parser
   - cypcb-router: FreeRouting integration (DSN/SES format)
   - cypcb-platform: Platform abstraction facade (native/web)
   - cypcb-calc: Calculation utilities
   - cypcb-watcher: File watching for live reload
   - cypcb-cli: CLI interface

4. **Data Flow:**
   - .cypcb source -> parser -> AST -> world (ECS) -> render
   - Edit cycle: editor -> debounce -> parse -> update world -> re-render
   - Export pipeline: world -> Gerber/drill generation

5. **Frontend Architecture:**
   - Vite + TypeScript
   - WASM module loading
   - Monaco editor integration
   - Theme system (ThemeManager)
   - Platform detection (desktop vs web)

Read each crate's Cargo.toml and src/lib.rs to document accurate dependencies and key exports.
