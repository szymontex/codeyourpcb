<task type="auto">
  <name>Task 1: Create file-picker.ts utilities</name>
  <files>viewer/src/file-picker.ts</files>
  <action>
Create viewer/src/file-picker.ts with three utility functions:

1. `readFileAsText(file: File): Promise<string>` - Promise wrapper for FileReader
   - Create new FileReader
   - Set up onload to resolve with reader.result as string
   - Set up onerror to reject with reader.error
   - Call reader.readAsText(file)

2. `createFilePicker(accept: string, onFile: (file: File) => void): HTMLInputElement`
   - Create hidden input element (type="file", accept attribute, display="none")
   - Add change event listener that:
     - Gets file from input.files[0]
     - Calls onFile callback if file exists
     - Resets input.value = '' to allow re-selecting same file
   - Append to document.body
   - Return the input element (caller will call .click() to open dialog)

3. `setupDropZone(element: HTMLElement, onDrop: (file: File) => void): void`
   - Track dragCounter (number) to handle child element drag events
   - Add dragenter handler: preventDefault, increment counter, add 'drag-over' class
   - Add dragleave handler: decrement counter, if 0 remove 'drag-over' class
   - Add dragover handler: preventDefault, set dataTransfer.dropEffect = 'copy'
   - Add drop handler: preventDefault, reset counter, remove 'drag-over' class, call onDrop with file
   - Add window-level dragover/drop with preventDefault to prevent browser navigation

All functions should be exported. TypeScript strict mode compatible.
  </action>
  <verify>

    - TypeScript compiles: `cd viewer && npx tsc --noEmit`
  </verify>

    - readFileAsText wraps FileReader in Promise
    - createFilePicker creates hidden input with change handler
    - setupDropZone sets up all drag events with visual feedback class
    - Window-level handlers prevent accidental file navigation
  </done>
