  <action>
Extend the Tree-sitter grammar to support footprint definitions.

Add to grammar.js:

1. Add `footprint_definition` to top-level definition alternatives.

2. Create footprint_definition rule:
   ```javascript
   footprint_definition: $ => seq(
     'footprint',
     $.identifier,
     '{',
     repeat($.footprint_property),
     '}'
   ),
   ```

3. Create footprint_property choice:
   ```javascript
   footprint_property: $ => choice(
     $.description_property,   // description "text"
     $.pad_definition,         // pad N shape at X, Y size W x H
     $.courtyard_property,     // courtyard W x H (optional)
   ),
   ```

4. Create pad_definition rule:
   ```javascript
   pad_definition: $ => seq(
     'pad',
     $.number,                 // Numeric pad ID (1, 2, 3...)
     $.pad_shape,              // rect, circle, roundrect, oblong
     'at',
     $.dimension,              // X position
     ',',
     $.dimension,              // Y position
     'size',
     $.dimension,              // Width
     'x',
     $.dimension,              // Height
     optional($.drill_spec),   // Optional: drill Nmm
   ),

   drill_spec: $ => seq('drill', $.dimension),

   pad_shape: $ => choice('rect', 'circle', 'roundrect', 'oblong'),
   ```

5. Create courtyard_property rule:
   ```javascript
   courtyard_property: $ => seq(
     'courtyard',
     $.dimension,
     'x',
     $.dimension
   ),
   ```

6. Regenerate parser: `cd grammar && npx tree-sitter generate`

Target syntax (from 03-RESEARCH.md):
```cypcb

    pad 2 rect at -2.7mm, -0.635mm size 1.5mm x 0.6mm
    // ...
    courtyard 6mm x 5mm
}
```

2. Add `convert_footprint_def()` helper function:
   ```rust
   fn convert_footprint_def(fp_def: &ast::FootprintDef) -> Footprint {
       let pads: Vec<PadDef> = fp_def.pads.iter().map(|p| {
           PadDef {
               number: p.number.to_string(),
               shape: convert_pad_shape(p.shape),
               position: Point::new(p.x.to_nm(), p.y.to_nm()),
               size: (p.width.to_nm(), p.height.to_nm()),
               drill: p.drill.map(|d| d.to_nm()),
               layers: if p.drill.is_some() {
                   vec![Layer::TopCopper, Layer::BottomCopper]
               } else {
                   vec![Layer::TopCopper, Layer::TopPaste, Layer::TopMask]
               },
           }
       }).collect();

       // Calculate bounds and courtyard from pads
       let bounds = calculate_bounds(&pads);
       let courtyard = fp_def.courtyard
           .map(|(w, h)| Rect::centered(Point::ORIGIN, w.to_nm(), h.to_nm()))
           .unwrap_or_else(|| bounds.expand(Nm::from_mm(0.5))); // IPC-7351B margin

       Footprint {
           name: fp_def.name.0.clone(),
           description: fp_def.description.clone().unwrap_or_default(),
           pads,
           bounds,
           courtyard,
       }
   }
   ```

3. Add `convert_pad_shape()` helper:
   ```rust
   fn convert_pad_shape(shape: ast::PadShape) -> PadShape {
       match shape {
           ast::PadShape::Rect => PadShape::Rect,
           ast::PadShape::Circle => PadShape::Circle,
           ast::PadShape::RoundRect => PadShape::RoundRect,
           ast::PadShape::Oblong => PadShape::Oblong,
       }
   }
   ```
