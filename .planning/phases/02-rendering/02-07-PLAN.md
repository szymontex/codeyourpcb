<task type="checkpoint:human-verify" gate="blocking">
  <what-built>
Complete minimal PCB viewer with:
- WASM-based board parsing (cypcb-render crate)
- Canvas 2D rendering of board, components, pads
- Zoom/pan navigation (scroll wheel, middle-drag)
- Layer visibility toggles (top/bottom copper)
- Component selection (click to select)
- Hot reload on file save
  </what-built>
  <how-to-verify>
1. Start the development server:
   ```
   cd viewer
   npm run build:wasm
   npm run dev:watch
   ```

2. Open browser at http://localhost:5173

3. Verify rendering:
   - Board outline visible (yellow rectangle)
   - Component pads visible (red for top copper)
   - Test with examples/blink.cypcb

4. Verify navigation:
   - Scroll wheel zooms (should zoom toward cursor position)
   - Middle-click + drag pans the view
   - Coordinates update in toolbar as mouse moves

5. Verify layer toggles:
   - Uncheck "Top" checkbox - top copper pads should disappear
   - Check it again - pads return
   - Same for "Bottom" checkbox

6. Verify selection:
   - Left-click on a pad area
   - Component pads should turn orange (selected)
   - Click elsewhere to deselect

7. Verify hot reload:
   - In another terminal, edit examples/blink.cypcb
   - Change a component position (e.g., change "at 10mm, 15mm" to "at 15mm, 20mm")
   - Save the file
   - Viewer should update within ~500ms
   - Zoom/pan position should be preserved
   - Status should briefly show "Reloaded"

8. Verify error handling:
   - Edit blink.cypcb to have invalid syntax
   - Save - should show error in status or console
   - Fix syntax - should recover and render again
  </how-to-verify>
  <resume-signal>
Type "verified" if all checks pass, or describe any issues found.
  </resume-signal>
