1. Version declaration (`version 1`)
2. Board definition (`board name { size, layers }`)
3. Component definition with all properties:
   - `component REFDES kind "footprint" { value, at, rotation, pin.X = NET }`
4. Net definition with constraint syntax:
   - Basic: `net NAME { pins }`
   - With constraints: `net NAME [constraint1, constraint2] { pins }`
   - Available constraints: `width`, `clearance`, `current`
5. Zone definition (`zone name { bounds, layer }`, `keepout name { bounds }`)
6. Trace definition (`trace NET { from, to, layer, width, locked }`)
7. Custom footprint definition

IMPORTANT: Emphasize that net constraints use square brackets BEFORE the braces:
```
// CORRECT - constraints in square brackets before braces
net VCC [current 500mA] {
    R1.1
    C1.1
}

// INCORRECT - constraints cannot go inside braces
net VCC {
    R1.1
    current 500mA  // ERROR: unexpected token
}
```

Include examples for each section. Reference the example files in examples/.
