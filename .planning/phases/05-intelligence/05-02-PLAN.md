  <action>
Implement IPC-2221 trace width calculation:

1. **TraceWidthParams struct**:
   - current_amps: f64 (required)
   - temp_rise_c: f64 (default 10.0)
   - copper_oz: f64 (default 1.0, range 0.5-3.0)
   - is_external: bool (default true, external vs internal layer)
   - ambient_temp_c: f64 (default 25.0, for info only)

2. **TraceWidthResult struct**:
   - width: Nm (calculated minimum width)
   - cross_section_mm2: f64 (for reference)
   - warnings: Vec<TraceWidthWarning>

3. **TraceWidthWarning enum**:
   - CurrentTooHigh (>35A formula accuracy degrades)
   - TempRiseTooLow (<10C may not be achievable)
   - TempRiseTooHigh (>100C risk of delamination)
   - WidthExceedsMax (>10mm, consider multiple traces)
   - CopperWeightNonStandard (not 0.5, 1.0, 2.0oz)

4. **TraceWidthCalculator**:
   - calculate(params: &TraceWidthParams) -> TraceWidthResult
   - Formula: I = k * dT^0.44 * A^0.725
   - Solve for A: A = (I / (k * dT^0.44))^(1/0.725)
   - Width = A / (thickness * 1.378)
   - k = 0.048 for external, 0.024 for internal

5. **Convenience methods**:
   - min_width_for_current(current_amps: f64, is_external: bool) -> Nm
   - with_defaults() constructor

6. **Tests**:
   - 1A current, 1oz copper, 10C rise, external -> ~0.25mm width
   - 1A current, internal layer -> ~0.5mm (needs more width)
   - 10A current -> larger width, no warnings
   - 50A current -> warning for CurrentTooHigh
   - Verify against known IPC-2221 reference values

Include doc examples showing typical usage.
