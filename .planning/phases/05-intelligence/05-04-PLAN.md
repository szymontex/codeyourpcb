<task type="auto">
  <name>Task 2: Implement DSN Export</name>
  <files>
    crates/cypcb-router/src/dsn.rs
  </files>
  <action>
Implement Specctra DSN export per research (see Pattern 3 in RESEARCH.md):

1. **DsnExportError enum**:
   - IoError(std::io::Error)
   - MissingBoard
   - EmptyNet(String)

2. **export_dsn(world: &BoardWorld, output: &mut impl Write) -> Result<(), DsnExportError>**:
   Main entry point.

3. **DSN Structure** (per Specctra specification):
   ```
   (pcb "board_name"
     (parser
       (string_quote ")
       (space_in_quoted_tokens on)
     )
     (resolution mil 10)
     (unit mil)
     (structure ...)
     (placement ...)
     (library ...)
     (network ...)
     (wiring ...)
   )
   ```

4. **write_structure(world, output)**: Board boundary and layers
   - Board boundary from BoardSize
   - Layer definitions (signal layers only)
   - Default design rules from DesignRules

5. **write_placement(world, output)**: Component positions
   - Iterate all entities with RefDes + Position
   - Output: (component "footprint" (place "refdes" x y side rotation))
   - Convert coordinates: nm -> mil (divide by 25400)
   - Side: front or back based on layer

6. **write_library(world, output)**: Footprints and padstacks
   - For each unique footprint, output (image ...)
   - Pad definitions as (padstack ...)
   - Pin positions relative to component origin

7. **write_network(world, output)**: Nets and connections
   - For each net: (net "name" (pins refdes-pin refdes-pin ...))
   - Net class with rules (width, clearance from constraints)

8. **write_wiring(world, output)**: Existing traces (locked only)
   - If Trace entities exist with locked=true, export as fixed wires
   - FreeRouting will not modify these

9. **Coordinate conversion helpers**:
   - nm_to_mil(nm: i64) -> f64 (nm / 25400)
   - Layer to DSN layer name ("Top" -> "F.Cu")

10. **Tests**:
    - Export empty board
    - Export board with components, verify DSN syntax valid
    - Export with nets, verify pin connections
    - Export with locked traces
  </action>

  </verify>
  <done>
    DSN export produces valid Specctra format.
    All board elements (boundary, components, nets) included.

  </done>
