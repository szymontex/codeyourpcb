<task type="auto">
  <name>Task 2: Implement Footprint Import</name>
  <files>
    crates/cypcb-kicad/src/footprint.rs
  </files>
  <action>
Implement KiCad .kicad_mod to internal Footprint conversion:

1. **KicadImportError enum**:
   - IoError(std::io::Error)
   - ParseError(String)
   - UnsupportedFeature(String)
   - MissingData(String)

2. **import_footprint(path: &Path) -> Result<Footprint, KicadImportError>**:
   - Use kicad_parse_gen::read_module(path)
   - Convert Module to internal Footprint

3. **Pad conversion** (convert_pad):
   - Shape mapping:
     - kicad Rect -> PadShape::Rect
     - kicad Circle/RoundRect -> PadShape::Circle/RoundRect
     - kicad Oval -> PadShape::Oblong
   - Position: Point::from_mm(at.x, at.y)
   - Size: (Nm::from_mm(size.x), Nm::from_mm(size.y))
   - Drill: p.drill.map(|d| Nm::from_mm(d.size))
   - Layer conversion:
     - F.Cu -> TopCopper
     - B.Cu -> BottomCopper
     - F.Paste -> TopPaste
     - F.Mask -> TopMask
     - etc.

4. **Handle THT vs SMD**:
   - Has drill -> THT, layers = TopCopper | BottomCopper
   - No drill -> SMD, layers from KiCad layers field

5. **Courtyard extraction**:
   - Look for fp_line elements on F.CrtYd/B.CrtYd layer
   - Calculate bounding box if present
   - Fall back to pad bounds + 0.5mm margin (IPC-7351B)

6. **Handle edge cases**:
   - Custom shaped pads (log warning, use bounding rect)
   - Negative offsets (fully supported)
   - Multiple footprint layers

7. **Tests with real KiCad footprints** (create test fixtures):
   - SMD resistor (0402, 0603)
   - Through-hole resistor
   - IC package (SOIC-8)
   - Connector with mixed THT pads
