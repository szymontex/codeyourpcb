  <action>
Implement copper layer Gerber export:

1. Define export_copper_layer function:
   ```rust
   pub fn export_copper_layer(
       world: &mut BoardWorld,
       library: &FootprintLibrary,
       layer: Layer,
       format: &CoordinateFormat,
   ) -> Result<String, ExportError>
   ```

2. Implement pad export:
   - Query all components with Position, FootprintRef, Rotation
   - For each component, look up footprint in library
   - For each pad on the target layer:
     - Calculate absolute position (component pos + rotated pad offset)
     - Get or create aperture for pad shape
     - Generate D{nn}* (select aperture)
     - Generate X{x}Y{y}D03* (flash at position)
   - Handle rotation: rotate pad position around component center

3. Implement trace export:
   - Query all Trace components on target layer
   - For each trace:
     - Get or create circular aperture matching trace width
     - Generate D{nn}* to select aperture
     - For first segment: X{x}Y{y}D02* (move to start)
     - For subsequent points: X{x}Y{y}D01* (draw to point)

4. Implement via export on copper layer:
   - Query all Via components
   - For vias that span this layer:
     - Get or create circular aperture for via outer_diameter
     - Flash via as circle at position

5. Assemble complete file:
   - Header (from header.rs)
   - Aperture definitions (from ApertureManager)
   - Drawing commands (pads, traces, vias)
   - M02* end of file

6. Add tests:
   - Test simple board with 2 pads
   - Test board with traces
   - Test via rendering
   - Verify output is valid Gerber structure

AVOID: Exporting to wrong layer - carefully check layer matching for pads/traces.

1. Create crates/cypcb-export/src/gerber/mask.rs
2. Update mod.rs to include `pub mod mask;`

3. Implement export_soldermask function:
   - Parameters: world, library, side (Top/Bottom), format
   - Soldermask is NEGATIVE - openings where no mask
   - Query components on specified side
   - For each pad:
     - Create aperture slightly larger than pad (mask expansion, typically 0.05mm)
     - Flash aperture at pad position (creates mask opening)
   - Include via mask openings (via outer_diameter + expansion)
   - Set %LP{D}*% (Dark polarity - exposed areas)

4. Implement export_solderpaste function:
   - Parameters: world, library, side (Top/Bottom), format
   - Only SMD pads get paste (check pad.drill.is_none())
   - Paste apertures may be slightly smaller (paste reduction, typically 0-10%)
   - For each SMD pad on side:
     - Create aperture at paste size
     - Flash at pad position

5. Add configuration struct for mask/paste parameters:
   ```rust
   pub struct MaskPasteConfig {
       pub mask_expansion: Nm,  // Default 50_000 nm (0.05mm)
       pub paste_reduction: f64, // Default 0.0 (0%)
   }
   ```

6. Add tests:
   - Test mask layer has openings for all pads
   - Test paste layer only has SMD pads
   - Test THT pads excluded from paste

Note: Soldermask is tricky - the "dark" areas are where solder mask EXISTS (covers copper), openings are where mask is ABSENT (pad exposed for soldering).
