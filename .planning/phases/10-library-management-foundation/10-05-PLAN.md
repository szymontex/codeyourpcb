  <action>
Create `manager.rs` with:

`LibraryManager` struct with fields:
- `conn: Arc<Mutex<Connection>>` - shared SQLite connection
- `kicad_source: KiCadSource` - KiCad parser
- `custom_source: CustomSource` - custom library manager
- Optional JLCPCB: `#[cfg(feature = "jlcpcb")] jlcpcb_source: Option<JLCPCBSource>`

Constructor:
- `pub fn new(db_path: &Path) -> Result<Self, LibraryError>`:
  - Open SQLite connection at db_path (or ":memory:" for testing)
  - Call `schema::initialize_schema(&conn)` to create tables
  - Initialize KiCadSource with empty search paths (configured later)
  - Initialize CustomSource with Arc clone of connection
  - Return Self

Configuration methods:
- `pub fn set_kicad_search_paths(&mut self, paths: Vec<PathBuf>)`: Update KiCad source search paths (LIB-10)
- `pub fn add_kicad_search_path(&mut self, path: PathBuf)`: Add single search path
- `#[cfg(feature = "jlcpcb")] pub fn configure_jlcpcb(&mut self, api_key: String)`: Enable JLCPCB source

Import operations:
- `pub fn import_kicad_library(&self, name: &str) -> Result<usize, LibraryError>`:
  - Call kicad_source.import_library(name) to get Vec<Component>
  - Insert library record via schema::insert_library
  - Batch insert components via schema::insert_components_batch
  - Return count of imported components
- `pub fn auto_import_folder(&self, path: &Path) -> Result<Vec<String>, LibraryError>`:
  - Call kicad_source.auto_organize_folder(path)
  - For each discovered library, import it
  - Return list of imported library names (LIB-11)

Search operations:
- `pub fn search(&self, query: &str, filters: &SearchFilters) -> Result<Vec<SearchResult>, LibraryError>`:
  - Lock connection
  - Call search::search_components with query and filters
  - Return results (searches across ALL indexed sources)
- `pub fn search_by_field(&self, field: &str, value: &str, limit: usize) -> Result<Vec<SearchResult>, LibraryError>`

Library management:
- `pub fn list_libraries(&self) -> Result<Vec<LibraryInfo>, LibraryError>`: List all libraries from all sources
- `pub fn list_kicad_libraries(&self) -> Result<Vec<LibraryInfo>, LibraryError>`: Scan KiCad search paths for available .pretty folders
- `pub fn create_custom_library(&self, name: &str) -> Result<LibraryInfo, LibraryError>`: Delegate to CustomSource
- `pub fn delete_library(&self, source: &str, name: &str) -> Result<(), LibraryError>`: Delete library and its components

Component access:
- `pub fn get_component(&self, source: &str, name: &str) -> Result<Option<Component>, LibraryError>`
- `pub fn component_count(&self) -> Result<usize, LibraryError>`

For testing, add:
- `pub fn new_in_memory() -> Result<Self, LibraryError>`: Creates manager with in-memory SQLite

Update `lib.rs` to add `pub mod manager;` and re-export `LibraryManager`.
