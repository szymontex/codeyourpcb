- `pub fn update_component_category(&self, name: &str, category: &str) -> Result<(), LibraryError>`: Update category for organization by function.
- `pub fn update_component_manufacturer(&self, name: &str, manufacturer: &str) -> Result<(), LibraryError>`: Update manufacturer for organization.
- `pub fn delete_library(&self, name: &str) -> Result<(), LibraryError>`: Delete library and all its components.

Add `pub mod custom;` to sources/mod.rs.

  <action>
Add to Cargo.toml:
- reqwest = { version = "0.12", features = ["json"], optional = true }
- Create feature `jlcpcb` that enables reqwest

Create `sources/jlcpcb.rs` (behind `#[cfg(feature = "jlcpcb")]`):

`JLCPCBSource` struct with fields:
- `client: reqwest::Client`
- `api_key: String`
- `base_url: String` (default "https://api.jlcpcb.com")

Constructor `new(api_key: String) -> Self` with default base_url.

Implement `LibrarySource` for JLCPCBSource:
- `source_name()` returns `"jlcpcb"`
- `list_libraries()`: Returns single LibraryInfo { source: "jlcpcb", name: "JLCPCB Parts", ... } since JLCPCB is one massive catalog.
- `import_library()`: Returns `Err(LibraryError::NotSupported("JLCPCB catalog is too large for full import. Use search instead."))`.

Additional async method:
- `pub async fn search_api(&self, query: &str, page: usize, page_size: usize) -> Result<Vec<Component>, LibraryError>`:
  - Send GET to `{base_url}/components/search` with query params: keyword, page, pageSize
  - Header: Authorization: Bearer {api_key}
  - Parse JSON response into JLCPCBResponse struct (internal)
  - Convert to Vec<Component> with source="jlcpcb"
  - On HTTP error, return LibraryError::ApiError with status code and message
  - On network error, return LibraryError::ApiError

Define internal structs for JSON deserialization:
- `JLCPCBResponse { data: Vec<JLCPCBComponent>, total: usize }`
- `JLCPCBComponent` with serde rename attributes matching JLCPCB API field names (componentCode, componentName, etc.) - see research examples

Add `#[cfg(feature = "jlcpcb")] pub mod jlcpcb;` to sources/mod.rs.

IMPORTANT: JLCPCB API requires manual application approval. This feature is optional and disabled by default. The code should compile and the rest of the library system should work perfectly without it.
