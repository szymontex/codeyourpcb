```typescript
/** Matches monaco.editor.IStandaloneThemeData shape. Defined locally to avoid monaco-editor dependency. */
interface MonacoThemeData {
  base: 'vs' | 'vs-dark';
  inherit: boolean;
  rules: Array<{ token: string; foreground?: string; background?: string; fontStyle?: string }>;
  colors: Record<string, string>;
}
```

- colors: editor chrome that matches the CSS variable palette from colors.css

  - editor.foreground: #1a1a1a

  - editorLineNumber.foreground: #999999
  - editorGutter.background: #f5f5f5
  - editor.selectionBackground: #add6ff
  - editorCursor.foreground: #000000

- Subscribes to themeManager so future changes auto-switch: `themeManager.subscribe((resolved) => { monaco.editor.setTheme(resolved === 'dark' ? 'cypcb-dark' : 'cypcb-light'); })`
